# MusB Diagnostics Website

A premium healthcare diagnostics website redesign featuring a modern "Medical Orange Premium" design system.

## Features
- **Redesigned Hero Section**: Full-width background image with bold, left-aligned typography.
- **Wellness Panels**: Categorized health panels with clear pricing and iconography.
- **Test Search**: Easy search for individual diagnostic tests.
- **Service Highlights**: Quick overview of value-added services like Mobile Phlebotomy and Self-Pay options.
- **Responsive Design**: Optimized for all screen sizes.

## How to View
1. Clone the repository.
2. Open `index.html` in any modern web browser.

## Technologies Used
- HTML5
- Tailwind CSS (via CDN)
- Google Fonts (Inter, Poppins)
- Material Symbols (Iconography)
